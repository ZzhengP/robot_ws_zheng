\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classMPC__Task}{M\+P\+C\+\_\+\+Task}} class define the cost (objectif) function for Model Predictive Control. 

And the related enlarged state, such as Px, Pu, Pxdq, Pudq

$ q_{1:N} = Px*q_{0} + Pu*U_{0:N-1} \\ dq_{1:N} = Pxdq*q_{0} + Pudq*U_{0:N-1} $
